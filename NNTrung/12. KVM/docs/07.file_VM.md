# Tổ chức file trong KVM
### 1. Thư mục lưu các disk của VM
```bash
/var/lib/libvirt/images
```
![altimage](../images/Screenshot_21.png)

### 2. Thư mục chứa các file `.xml` thông số kỹ thuật của VM
```bash
/etc/libvirt/qemu
```
![altimage](../images/Screenshot_22.png)
### 3. Thư mục chứa các file liên quan đến `network`
```bash
/etc/libvirt/qemu/networks
```
![altimage](../images/Screenshot_23.png)
### 4. Thư mục chứa các storage
```bash
/etc/libvirt/storage
```
![altimage](../images/Screenshot_24.png)

### 5. Thư mục lưu các bản snapshot của VM
```bash
/var/lib/libvirt/qemu/snapshot
```
![altimage](../images/Screenshot_25.png)
