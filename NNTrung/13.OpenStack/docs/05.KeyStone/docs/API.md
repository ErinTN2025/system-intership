# Tìm hiểu về API và sử dụng `curl` để gọi API
## 1. Khái niệm
- API: viết tắt là Application Programming Interface - giao diện lập trình ứng dụng
- API là các phương thức, giao thức kết nối với các thư viện và ứng dụng khác.
- API cung cấp khả năng truy xuất đến một tập các hàm hay dùng. Và từ đó có thể trao đổi dữ liệu giữa các ứng dụng.
