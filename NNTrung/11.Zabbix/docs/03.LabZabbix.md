# Triển khai Zabbix agent trên ubuntu 24.04
## 1. Chuẩn bị hệ thống
```bash
sudo apt update && sudo apt upgrade -y
sudo timedatectl set-timezone Asia/Ho_Chi_Minh
timedatectl status
```
## 2. Cài Zabbix Repository
```bash
wget https://repo.zabbix.com/zabbix/7.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_7.0-2+ubuntu24.04_all.deb
sudo dpkg -i zabbix-release_7.0-2+ubuntu24.04_all.deb
sudo apt update
```
## 3. Cài Zabbix Server, Frontend, Agent
```bash
sudo apt install -y zabbix-server-mysql \
                    zabbix-frontend-php \
                    zabbix-apache-conf \
                    zabbix-sql-scripts \
                    zabbix-agent2

```

## 4. Cài và cấu hình MySQL
```bash
# Cài MySQL
sudo apt install -y mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql

# Tạo database và user cho Zabbix
sudo mysql -uroot -p
```
Trong MySQL chạy:
```sql
CREATE DATABASE zabbix CHARACTER SET utf8mb4 COLLATE utf8mb4_bin;
CREATE USER 'zabbix'@'localhost' IDENTIFIED BY 'StrongPass@123';
GRANT ALL PRIVILEGES ON zabbix.* TO 'zabbix'@'localhost';
SET GLOBAL log_bin_trust_function_creators = 1;
FLUSH PRIVILEGES;
EXIT;
```
Import schema Zabbix vào database
```bash
zcat /usr/share/zabbix-sql-scripts/mysql/server.sql.gz | mysql \
  --default-character-set=utf8mb4 \
  -uzabbix -p zabbix
```

Nếu bị lỗi `ctrl z` hoặc máy bị đơ lag làm mất gói, có thể cài lại:
```bash
sudo mysql -uroot -e "
sudo mysql -uroot -e "
DROP DATABASE zabbix;
CREATE DATABASE zabbix CHARACTER SET utf8mb4 COLLATE utf8mb4_bin;
GRANT ALL PRIVILEGES ON zabbix.* TO 'zabbix'@'localhost';
SET GLOBAL log_bin_trust_function_creators = 1;
FLUSH PRIVILEGES;"
"
```
và 
```bash
sudo mysql -uroot -e "SHOW TABLES IN zabbix;" | wc -l
```
- Nó phải in ra số `0`

Sau khi xong
```bash
# Kiểm tra có lỗi không
cat /tmp/zabbix_import.log

# Đếm số tables
sudo mysql -uroot -e "
SELECT COUNT(*) as total_tables 
FROM information_schema.tables 
WHERE table_schema = 'zabbix';"
```
- Số tables phải lớn hơn 170 nếu thấp hơn là thiếu

Sau khi import xong, tắt `log_bin_trust_function_creators`:
```bash
sudo mysql -uroot -p -e \
  "SET GLOBAL log_bin_trust_function_creators = 0;"

```
## 5. Cấu hình Zabbix Server
```bash
sudo nano /etc/zabbix/zabbix_server.conf
```
Chỉnh sửa như sau:
```ini
DBHost=localhost
DBName=zabbix
DBUser=zabbix
DBPassword=StrongPass@123

# Tuning cơ bản (tùy RAM)
StartPollers=10
StartPingers=5
CacheSize=128M
HistoryCacheSize=64M
```
## 6. Cấu hình Zabbix Agent 2
```bash
sudo nano /etc/zabbix/zabbix_agent2.conf
```
```ini
Server=127.0.0.1
ServerActive=127.0.0.1
Hostname=zabbix-server     # phải khớp với tên host trong Web UI
```

## 7. Khởi động các service
```bash
sudo systemctl restart zabbix-server zabbix-agent2 apache2
sudo systemctl enable zabbix-server zabbix-agent2 apache2

# Kiểm tra trạng thái
sudo systemctl status zabbix-server
sudo systemctl status zabbix-agent2
```

## 8. Mở firewall
```bash
sudo ufw allow 80/tcp      # Web UI
sudo ufw allow 443/tcp     # HTTPS (nếu dùng SSL)
sudo ufw allow 10050/tcp   # Zabbix Agent (passive)
sudo ufw allow 10051/tcp   # Zabbix Server
sudo ufw reload
```
Kiểm tra sau cài đặt
```bash
# Xem log server
sudo tail -f /var/log/zabbix/zabbix_server.log

# Xem log agent
sudo tail -f /var/log/zabbix/zabbix_agent2.log

# Test kết nối agent từ server
zabbix_get -s 127.0.0.1 -p 10050 -k "system.cpu.util"
```
